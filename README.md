# 🤖 AI Chatbot Website

An AI-powered chatbot web application built with **Python**, **Flask**, and **Google Gemini AI**. Users can register, log in, and chat with an AI assistant through a modern web interface.

---

## 🚀 Features

- 🔐 User Registration & Login
- 🤖 Google Gemini AI Integration
- 💬 Real-time Chat Interface
- 📝 Chat History
- 🎨 Modern Responsive UI
- 🔒 Secure Environment Variables
- 🗄️ SQLite Database Support

## ⚙️ Installation

### 1. Clone the repository

```bash
git clone https://github.com/Riya0Dhiman123/ai-chatbot-website.git
```

### 2. Open the project

```bash
cd ai-chatbot-website
```

### 3. Install dependencies

```bash
pip install -r requirements.txt
```

### 4. Create a `.env` file

```env
GEMINI_API_KEY=YOUR_GEMINI_API_KEY
SECRET_KEY=YOUR_SECRET_KEY
```

### 5. Run the application

```bash
python app.py
```

Open your browser and visit:

```
http://127.0.0.1:5000
```
